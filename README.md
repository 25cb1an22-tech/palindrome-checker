# Palindrome Checker Using Stack

## overview
Palindrome Checker Game is a simple web-based application that checks whether a word or number reads the same forward and backward. The project demonstrates basic programming logic using an interactive browser interface.

## Description
This project allows users to enter a word or number and determines whether it is a palindrome. The logic compares characters from the beginning and end of the string to verify if they match. The application is designed as a simple interactive game using HTML and JavaScript.

## Features
-Checks if a word or number is a palindrome

-Simple and interactive interface

-Fast result display

-Lightweight web project

## Language
HTML and JavaScript

## Project Structure
index.html – Main starting page

menu.html – Game menu interface

game.html – Game interaction page

new.js – JavaScript logic for palindrome checking

## How to Run
1.Download or clone the repository

2.Open index.html in your browser

3.Enter a word or number

4.The program will show whether it is a palindrome
