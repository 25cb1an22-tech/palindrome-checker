# Palindrome Checker Using Stack

## Description
This project checks whether a string is a palindrome using the stack data structure.

## Features
- Uses stack logic
- Accepts user input
- Displays result

## Language
HTML and JavaScript

## Author
Anushka Dhakad

## How to Run

1. Download the project
2. Open index.html in a browser
3. Start the program
4. Check if the word is a palindrome
